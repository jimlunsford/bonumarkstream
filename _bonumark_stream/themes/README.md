# Bonumark Stream Themes

Bonumark Stream themes are code-free presentation packages.

Use `_bonumark_stream/themes/default/` as the working example. Copy the Midnight Ledger folder, rename it, update `theme.json`, replace the screenshot, edit the CSS, zip the folder, and upload it from Appearance.

Themes may include metadata, settings, CSS, images, fonts, screenshots, and docs. Themes may not include PHP, JavaScript, HTML files, route handlers, database writes, permission logic, server config files, symlinks, or application behavior.
